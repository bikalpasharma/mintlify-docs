Welcome to the Component Architecture Documentation. This documentation covers all the available components and how to use them in your application.

## Overview

This documentation provides comprehensive information about our component system, including:

- **Program Header** - Banner images with configurable button containers
- **Program Details** - Flexible detail cards with nested component support
- **Multiple Cards** - Container for combining multiple card components
- **Component Reference** - Complete reference of all available components
